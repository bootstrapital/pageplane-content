Pageplane Content

This repository publishes structured content for Pageplane using Flatcontent.

Available artifacts

  • Content JSON: content.json
  • Collections: blog, docs, changelog

What this is

Markdown files are compiled into a single JSON artifact via flatcontent and hosted on GitHub Pages.

Pageplane (or any static HTML frontend) can fetch this JSON at runtime or during a build step.

What this is not

This is not a CMS UI or a website. It is a content source of truth.

Built with flatcontent. Content lives in this repo. Presentation lives elsewhere.
